# Screenshot Checklist — AccessFlow Wix App

Capture these screenshots once the app runs on a test Wix site. Use them for the Wix App Market listing and marketing.

## Required screenshots

1. **Floating button on site**  
   The accessibility button visible in the bottom-right corner of a Wix site homepage (or your chosen position).

2. **Open toolbar panel**  
   Full panel open showing all control groups: text size, spacing, font, contrast, color filter, reading & motion toggles, and Reset.

3. **Dark mode applied**  
   Same Wix page with AccessFlow dark contrast mode enabled — visible background and text change across the page.

4. **Dyslexia-friendly font**  
   A content-heavy Wix page with OpenDyslexic (or dyslexia-friendly font) applied site-wide via AccessFlow.

5. **Dashboard settings page**  
   AccessFlow Settings inside Wix admin showing Toolbar Appearance, Visible Features grid, and Save button with Active badge.

## Optional screenshots

6. **Accent color customized** — dashboard color picker + matching magenta (or custom) active states on the toolbar.

7. **Feature hidden** — dashboard with a feature toggled off + frontend panel missing that section.

8. **Mobile view** — floating button and panel on a narrow viewport.

## Tips

- Use a clean Wix template with readable body text.
- Hide unrelated browser chrome; crop to 1280×800 or Wix’s recommended dimensions.
- Do not include personal or student data in screenshots.
- Filename examples: `01-floating-button.png`, `02-panel-open.png`, etc.
