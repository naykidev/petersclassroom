# AccessFlow

Visitor-controlled accessibility toolbar by **Axolo Assist**.

## Install

### npm (recommended)

```bash
npm install @axolassist/accessflow
```

```ts
import '@axolassist/accessflow';
```

### CDN

```html
<script src="https://cdn.axolassist.com/accessflow.js"></script>
```

## Source layout

| Path | Purpose |
|------|---------|
| `packages/accessflow/` | TypeScript SDK (`@axolassist/accessflow`) |
| `cdn/` | CDN IIFE bundle (synced on `npm run build`) |
| `accessflow/demo.html` | Local script-tag demo |

See [`packages/accessflow/README.md`](../packages/accessflow/README.md) for full documentation.
