# AccessFlow

Visitor-controlled accessibility toolbar by **Axolo Assist**.

## Install

### npm (recommended)

```bash
npm install @axol-assist/accessflow
```

```ts
import '@axol-assist/accessflow';
```

### CDN

```html
<script src="https://axolassist.com/cdn/accessflow.js"></script>
```

## Source layout

| Path | Purpose |
|------|---------|
| `packages/accessflow/` | TypeScript SDK (`@axol-assist/accessflow`) |
| `cdn/` | CDN IIFE bundle (synced on `npm run build`) |
| `accessflow/demo.html` | Local script-tag demo |

See [`packages/accessflow/README.md`](../packages/accessflow/README.md) for full documentation.
